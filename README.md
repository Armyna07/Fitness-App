#  Fitness Challenge Web Platform

A web app where users can join fitness challenges, compete with friends, and track progress on a leaderboard.

Built with the MERN stack (MongoDB, Express, React, Node.js).

---

##  Team Roles

| Member 2 | Backend/API — Express server & routes |
| Member 3 | Database — MongoDB schemas & models |
| Member 4 | Integration & PM — GitHub, testing, debugging |
